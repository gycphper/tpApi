ThinkPHP 5.1 —— JWT API  just for TESTING
===============

> ThinkPHP5的运行环境要求PHP5.6以上。

## 安装

使用composer安装

~~~
composer create-project topthink/think tp
~~~

启动服务

~~~
cd tp
php think run
~~~

然后就可以在浏览器中访问

~~~
http://localhost:8000
~~~

更新框架
~~~
composer update topthink/framework
~~~
